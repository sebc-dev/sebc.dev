- Pillar blocks: 1 col mobile, 3 cols desktop (Claude's discretion for responsive)
- CTA: Button-style link to home page (per locked decision "lien vers la home ou search")
- All sections use `fade-up` for scroll-reveal animation (existing BaseLayout mechanism)
- Profile photo: placeholder path `/images/profile.jpg` -- user will provide the actual file
